# Loop Challenges

Solved 5 JavaScript challenges using `for` loops, `while` loops, and `if` statements.

## Challenges Solved

1. **Odd numbers from 1 to 20**  
   Printed all odd numbers using a loop and condition.

2. **Multiples of 3 from 100 down to 0**  
   Used a loop to print numbers divisible by 3 in decreasing order.

3. **Sequence: 4, 2.5, 1, -0.5, -2, -3.5**  
   Used a `while` loop with a decreasing step of 1.5.

4. **Sigma (Sum 1 to 100)**  
   Calculated the sum of all numbers from 1 to 100. Result: 5050.

5. **Factorial (1 to 12)**  
   Multiplied all numbers from 1 to 12. Result: 479001600.


## What I Practiced

- `for` and `while` loops
- `if` conditions
- Mathematical operations
